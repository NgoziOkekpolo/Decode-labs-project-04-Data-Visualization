# Decode-labs-project-04-Data-Visualization with Microsoft Power BI

Overview

This project focuses on transforming raw e-commerce sales data into an interactive dashboard using Microsoft Power BI. The goal was to present business information in a visual format that makes it easier to understand sales performance, customer behavior, and product trends.

The dashboard was built using the same dataset from the previous SQL analysis project. Before creating the visuals, the data was reviewed to ensure it was clean and correctly formatted for analysis.

---

Project Objective

The objective of this project was to:

- Build an interactive sales dashboard in Power BI.
- Present important business metrics using KPI cards.
- Visualize sales performance with appropriate charts.
- Create filters that allow users to interact with the dashboard.
- Communicate business insights through data visualization.

---

Dataset Information

- Dataset: E-Commerce Sales Dataset
- Total Records: 1,200
- Total Columns: 14

The dataset contains information such as:

- Order ID
- Product
- Quantity
- Total Price
- Payment Method
- Order Status
- Referral Source
- Items in cart
- Unit Price
- Date
- Coupon Code
- Customer ID

---

Data Preparation

The dataset used for this project was already cleaned during previous internship projects.

Additional preparation included:

- Importing the cleaned dataset into Power BI.
- Confirming that each column had the correct data type.
- Formatting sales values as currency.
- Creating measures for Total Sales, Total Orders, and Average Order Value.

---

Key Insights

The dashboard revealed several useful business insights.

- Total sales generated were approximately $1.26 million.
- A total of 1,200 orders were processed.
- The average order value was approximately $1,053.97.
- Chair and Printer generated the highest sales among all products.
- Credit Card and Online payments contributed the highest sales.
- Instagram and Email were the strongest referral sources.
- Order statuses were fairly balanced across Delivered, Shipped, Pending, Returned, and Cancelled.
